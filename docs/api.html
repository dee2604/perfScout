Core Performance Metrics

getCpuInfo()

Retrieves current CPU usage and information.

Returns: CpuInfo object containing CPU usage percentage and core information

Example:

val cpuInfo = PerfScout.getCpuInfo() println("CPU Usage: ${cpuInfo.usagePercent}%") println("Cores: ${cpuInfo.coreCount}")

getRamInfo(context: Context)

Retrieves RAM usage and memory information.

Returns: RamInfo object containing total, available, and used memory

Example:

val ramInfo = PerfScout.getRamInfo(context) println("Total RAM: ${ramInfo.totalMemory} MB") println("Available: ${ramInfo.availableMemory} MB")

getBatteryInfo(context: Context)

Retrieves battery status and health information.

Returns: BatteryInfo object containing level, health, and charging status

Advanced Monitoring

setCrashListener(listener: (CrashInfo) -> Unit)

Sets up crash monitoring with callback notifications.

Example:

PerfScout.setCrashListener { crashInfo -> Log.e("PerfScout", "Crash detected: ${crashInfo.message}") // Send to crash reporting service Crashlytics.logException(crashInfo.throwable) }

setAnrListener(listener: (AnrInfo) -> Unit)

Sets up ANR (Application Not Responding) monitoring.

Example:

PerfScout.setAnrListener { anrInfo -> Log.w("PerfScout", "ANR detected: ${anrInfo.duration}ms") }

getFrameRenderingInfo(duration: Long, callback: (FrameRenderingInfo) -> Unit)

Analyzes frame rendering performance over a specified duration.

Parameter Type Description
duration Long Duration in milliseconds to analyze
callback (FrameRenderingInfo) -> Unit Callback with analysis results

Memory Management

monitorMemoryLeaks(context: Context, callback: (Boolean, List) -> Unit)

Monitors for potential memory leaks using heuristic analysis.

Example:

PerfScout.monitorMemoryLeaks(context) { isLeaking, usageHistory -> if (isLeaking) { Log.w("PerfScout", "Possible memory leak detected") // Analyze usage history for patterns usageHistory.forEach { usage -> println("Memory at ${usage.timestamp}: ${usage.usedMemory} MB") } } }

Startup Time Tracking

recordActivityOnCreate(activity: Activity)

Records the start time of activity creation for startup tracking.

Example:

class MainActivity : AppCompatActivity() { override fun onCreate(savedInstanceState: Bundle?) { super.onCreate(savedInstanceState) PerfScout.recordActivityOnCreate(this) // ... rest of onCreate } }

recordFirstDraw(activity: Activity)

Records the first draw time for accurate startup measurement.

getStartupTimeInfo()

Retrieves startup time information if tracking was set up.
